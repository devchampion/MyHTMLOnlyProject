This section talks about few playcodes that i have done and played with. Let me show you some of them

  • BLOCK ELEMENT: This going to be a brief note about block element.

    These set of elements consists of all headers, blockquote etc. They are called block element because they take their space and push every other tag around them away.

  • INLINE ELEMENT: Here is a brief note about inline element

    These is the opposite of what block element is. Rather than taking a whole new space for itself, it accomodates other element e.g p-tags, bold-tags etc.

  • STRONG & EPMPHASIS: There is no big/noticeable difference between the STRONG and the BOLD. I use strong most times brcause it is better for me.

    The Emphasisand italics has no difference but we use emphasis because it is shorter and kind of tells a beginner it's work from the first view.
